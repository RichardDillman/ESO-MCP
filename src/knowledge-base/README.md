# ESO MCP Knowledge Base

This directory contains accumulated knowledge about ESO data structures, scraping challenges, and solutions. This serves as a reference to avoid repeating mistakes and to understand the complexities of ESO game data.

## Purpose

- Document data structure insights
- Record scraping challenges and solutions
- Provide examples of complex data relationships
- Serve as a reference for future development

## Contents

- `mundus-stones.md` - Mundus stones data structure and Divines trait scaling
- `skills.md` - Skills, morphs, and progression systems
- `sets.md` - Item sets, bonuses, and scaling mechanics
